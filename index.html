SMART VOTING SYSTEM WITH FACE RECOGNITION


Technology Used - Python , Django , Flask , OpenCV , MYSql , HTML & CSS , JS
Developed a secure, smart voting system with facial recognition to enhance the accuracy, accessibility, and security of the voting process. The system uses Flask, OpenCV for facial recognition, and MySQL for data management, with Python and Django for the backend. Voter identity is verified through facial biometric data to prevent fraud. The system includes two account types: voters and organizers, with organizers able to create polls through a user-friendly interface. UML diagrams (use case, sequence, activity, and class diagrams) were employed for system design. This project aims to improve the voting process, ensuring free, fair, and efficient elections

DISEASE PREDICTION BASED ON SYMPTOMS


Technology Used - Python , Decision tree , Machine learning
Disease Prediction System using Machine Learning to predict diseases based on symptoms provided by users. Utilized the Decision Tree algorithm, a supervised learning technique, to classify and predict the likelihood of diseases such as Diabetes, Malaria, Jaundice, Dengue, and Tuberculosis. The system processes user input and outputs the probability of the disease, aiding early diagnosis and improving patient care. Implemented Linear Regression alongside decision trees for precise probability estimation. The project leveraged increasing biomedical and healthcare data to enhance accuracy in disease detection, ultimately benefiting healthcare providers and patients
